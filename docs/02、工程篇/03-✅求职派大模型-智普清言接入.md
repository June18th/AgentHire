官方已经介绍了哪些模型是免费可用的，对此有疑问的小伙伴可以直接到这里查看 [https://www.bigmodel.cn/pricing](https://www.bigmodel.cn/pricing)

## 一、ApiKey获取
首先进入官方开发者后台：[https://open.bigmodel.cn/](https://open.bigmodel.cn/)


登录成功之后，进入ApiKey管理页面 [https://open.bigmodel.cn/usercenter/proj-mgmt/apikeys](https://open.bigmodel.cn/usercenter/proj-mgmt/apikeys)，添加新的API Key，在弹窗中新建


![](https://cdn.nlark.com/yuque/0/2025/png/35158118/1753585898140-bbcb68d7-993c-4c81-bd08-b2e8853861b0.png)


保存完之后，在列表中将APIKey复制出来


## 二、项目配置
将上面获取的APIKey，拷贝到本地资源目录下 dev/application-ai.yml 文件中的 api-key 属性中


![](https://cdn.nlark.com/yuque/0/2025/png/35158118/1753586019933-044349c3-057e-4108-82b8-eab5a7a73a3e.png)


除了上面这个方案之外，我个人更推荐第二种方案，以启动参数的方式携带 apiKey，这样每次提交代码时，就不怕apiKey泄露了，具体的实现方案是：

+ 编辑启动配置
+ 在弹窗中，输入快捷键 alt + r (或者点击 Modify options，然后开启 program arguments输入框）
+ 在输入框中输入 `--zhipuai-api-key=` 等号后面的就是你的apiKey

## 三、扩展说明
SpringAI有实现非常完备的智谱清言的大模型的交互starter，如果你想在其他的项目中接入智谱，直接在pom.xml中导入依赖即可

```xml
<dependency>
    <groupId>org.springframework.ai</groupId>
    <artifactId>spring-ai-starter-model-zhipuai</artifactId>
</dependency>
```

对应的配置参数为

```yaml
spring:
  ai:
    zhipuai:
      # https://open.bigmodel.cn/usercenter/proj-mgmt/apikeys
      # api-key 使用你自己申请的进行替换；如果为了安全考虑，可以通过启动参数进行设置
      api-key: ${zhipuai-api-key}
      chat:
        options:
          model: GLM-4-Flash
```
