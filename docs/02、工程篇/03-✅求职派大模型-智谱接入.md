# 求职派大模型-智谱接入

接下来介绍如何获取 ApiKey 并在项目中完成 LLM 配置。

智谱模型列表可在这里查看：https://www.bigmodel.cn/pricing

![](https://cdn.paicoding.com/stutymore/03-✅求职派大模型-智谱接入-20260626152636.png)

可以用付费模型，也可以用免费模型。

## 一、ApiKey 获取

首先进入官方开发者后台：https://open.bigmodel.cn/

登录成功之后，进入 ApiKey 管理页面 https://open.bigmodel.cn/usercenter/proj-mgmt/apikeys，添加新的 API Key，在弹窗中新建：

![](https://cdn.paicoding.com/paicoding/b34259db97ddb0676cef58f8fff5c9c1.png)

![](https://cdn.paicoding.com/stutymore/01-✅求职派本地开发环境搭建指南-20260617170646.png)

保存完之后，在列表中将 ApiKey 复制出来。

## 二、项目配置

求职派的大模型供应商配置统一在后台维护，不再通过环境变量或 IDEA 启动参数配置 API Key。

启动后进入后台（用户头像点击【后台管理】），打开 **LLM供应商** 页面。

![](https://cdn.paicoding.com/stutymore/03-✅求职派大模型-智谱接入-20260626172134.png)

添加供应商，将刚刚复制的 ApiKey 填到 API Key 输入框里。

Base URL 填 `https://open.bigmodel.cn/api/paas/v4`，如果你有 Coding Plan 的话，填 `https://open.bigmodel.cn/api/coding/paas/v4`。

这样会直接从你的 Coding Plan 额度里扣费。

API 风格默认 OpenAI 就行，模型清单默认会有 `glm-4.7` 和 `glm-4.6v-flash`，如果你想使用其他模型，比如说 GLM-5.2 和 glm-5v-trubo，可以在模型清单里添加。

如果 IM 通道里出现“未找到 glm-4.7-flash 模型”，说明用户偏好或默认文本模型还指向旧模型名。进入后台 **LLM供应商** 页面确认智谱模型清单包含 `glm-4.7`，再把文本模型偏好设置为 `zhipu#glm-4.7`。

![](https://cdn.paicoding.com/stutymore/03-✅求职派大模型-智谱接入-20260626201455.png)

填完可以点一下【测试连接】，确认通不通。

![](https://cdn.paicoding.com/stutymore/03-✅求职派大模型-智谱接入-20260626202907.png)

保存后，配置会写入后端的全局配置表，后续模型调用会直接读取这份配置。
