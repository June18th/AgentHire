# 求职派大模型-智谱接入

求职派默认使用智谱的免费大模型：文本对话使用 `GLM-4.7-Flash`，视觉理解使用 `GLM-4.6V-Flash`。接下来介绍如何获取 ApiKey 并在项目中完成配置。

官方免费模型列表可在这里查看：[https://www.bigmodel.cn/pricing](https://www.bigmodel.cn/pricing)

![](https://cdn.nlark.com/yuque/0/2025/png/35158118/1753585772632-14fa28b2-7bc9-461c-9efa-e513bd6a2478.png)

## 一、ApiKey 获取

首先进入官方开发者后台：[https://open.bigmodel.cn/](https://open.bigmodel.cn/)

登录成功之后，进入 ApiKey 管理页面 [https://open.bigmodel.cn/usercenter/proj-mgmt/apikeys](https://open.bigmodel.cn/usercenter/proj-mgmt/apikeys)，添加新的 API Key，在弹窗中新建：

![](https://cdn.nlark.com/yuque/0/2025/png/35158118/1753585898140-bbcb68d7-993c-4c81-bd08-b2e8853861b0.png)

保存完之后，在列表中将 ApiKey 复制出来。

## 二、项目配置

### 方式一：通过 .env 文件配置（推荐）

项目根目录下有一个 `.env.example` 文件，首先将它复制为 `.env`：

```bash
cp .env.example .env
```

然后编辑 `.env`，找到 `ZHIPU_API_KEY` 这一行，将你的 ApiKey 填入：

```properties
ZHIPU_API_KEY=你的ApiKey
```

`.env` 文件已被 `.gitignore` 忽略，不会被提交到版本库，因此不用担心 ApiKey 泄露。

### 方式二：通过 IDEA 启动参数配置

如果你不想使用 `.env` 文件，也可以通过 IDEA 启动参数携带 ApiKey：

1. 编辑启动配置
2. 在弹窗中，输入快捷键 `Alt + R`（或者点击 Modify options，然后开启 Program arguments 输入框）
3. 在输入框中输入 `--ZHIPU_API_KEY=你的ApiKey`

## 三、模型偏好配置

项目通过 `provider#ModelName` 格式来指定使用哪个模型。默认配置在 `.env` 中：

```properties
# 文本对话模型
AGENT_AI_PREFERENCE_TEXT=zhipufree#GLM-4.7-Flash
# 视觉理解模型
AGENT_AI_PREFERENCE_VISION=zhipufree#GLM-4.6V-Flash
```

其中 `zhipufree` 是项目内置的 provider 名称，通过 OpenAI 兼容接口对接智谱。当前智谱提供的免费模型包括：

| 模型名称 | 类型 | 上下文 | 说明 |
|---------|------|--------|------|
| GLM-4.7-Flash | 文本 | 200K | 默认文本对话模型（推荐） |
| GLM-4-Flash-250414 | 文本 | 128K | GLM-4-Flash 更新版 |
| GLM-4-Flash | 文本 | 128K | 经典文本模型 |
| GLM-Z1-Flash | 推理 | 128K | 推理模型 |
| GLM-4.6V-Flash | 视觉 | - | 默认视觉理解模型，支持图片、视频（推荐） |
| GLM-4.1V-Thinking-Flash | 视觉 | 64K | 10B 级视觉推理模型 |
| GLM-4V-Flash | 视觉 | 4K | 经典图像理解模型 |

如果想切换模型，修改 `.env` 中的 `AGENT_AI_PREFERENCE_TEXT` 或 `AGENT_AI_PREFERENCE_VISION` 即可。

## 四、扩展说明

### 在其他项目中接入智谱

Spring AI 提供了智谱的官方 starter，如果你想在其他项目中接入智谱，直接在 `pom.xml` 中引入依赖：

```xml
<dependency>
    <groupId>org.springframework.ai</groupId>
    <artifactId>spring-ai-starter-model-zhipuai</artifactId>
</dependency>
```

对应的配置参数：

```yaml
spring:
  ai:
    zhipuai:
      api-key: ${zhipuai-api-key}
      chat:
        options:
          model: GLM-4.7-Flash
```

如果你希望从 0 到 1 创建一个 Spring AI 项目，不妨看看 [创建一个 Spring AI 的示例工程](https://www.yuque.com/itwanger/yyt72l/lcycogfotog4nlpf) 这篇文章。

### 接入没有官方 starter 的大模型

并不是所有的大模型都有 `spring-ai-starter`，对于那些没有的大模型，可以通过 OpenAI 兼容接口的方式接入。具体可以查看 AI 实战篇中的自定义大模型接入相关章节。
