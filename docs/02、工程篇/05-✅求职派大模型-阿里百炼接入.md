# 二、配置
使用上面我们申请的 apiKey/model 替换下面的配置

```java
spring:
  ai:
    dashscope:
      # 阿里的千帆大模型
      api-key: ${dashscope-api-key}
      chat:
        options:
          model: qwen-plus
```
